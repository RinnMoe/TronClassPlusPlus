Usage Guide

使用教程

安装 TronClass++ 后，在畅课学习活动页面打开扩展弹窗，即可使用课件下载与视频播放辅助功能。

01

下载扩展

从 GitHub Releases 下载最新版本并解压；如果暂未发布打包版本，也可以下载源码 ZIP 后解压。

02

Chromium 系安装

进入浏览器扩展管理页面，开启开发者模式，选择“加载未打包的扩展”，并选中包含 manifest.json 的扩展文件夹。

03

课件下载

打开畅课课程内容或学习活动页面，点击扩展图标。弹窗会读取当前页面可下载课件，并提供单个下载与下载全部。

04

视频播放辅助

在扩展弹窗切换到“视频播放”，可查看当前页面视频状态，并按需启用自动播放视频和播放完成后跳转下一项。
