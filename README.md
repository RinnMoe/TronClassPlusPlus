# TronClass++

TronClass++ 是一个畅课功能增强插件，支持仅预览文件下载、批量下载和视频播放辅助。

- 插件介绍页维护在 [`docs/index.html`](docs/index.html)
- 使用教程维护在 [`docs/tutorial.html`](docs/tutorial.html)
- 下载入口为 [GitHub Releases](https://github.com/RinnMoe/TronClassFileDownloadEnhancer/releases)

## 安装

1. 从 Releases 下载最新版本并解压；如果暂未发布打包版本，也可以下载源码 ZIP。
2. 在 Chromium 系浏览器中打开扩展管理页面，开启开发者模式。
3. 选择“加载未打包的扩展”，并选中包含 `manifest.json` 的扩展文件夹。

## 说明

本扩展理论上兼容所有畅课(TronClass)平台，可在 Chromium 系浏览器使用。
