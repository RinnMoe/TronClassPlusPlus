# TronClass++

TronClass++ 是一个畅课功能增强插件  
提供仅预览文件下载、批量下载和视频播放辅助等功能。


## 安装

请见[使用教程](https://www.doubao.com/thread/w673a6ddbec450a73)

## 说明

本扩展理论上兼容所有畅课(TronClass)平台，可在 Chromium 系浏览器使用。
